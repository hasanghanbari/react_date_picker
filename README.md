# React Persian Datepicker

This project provides a datepicker with Persian and Gregorian calendar support using React and Vite.

## Features:

- Datepicker with the ability to select dates using both Persian and Gregorian calendars.
- Utilizes React for dynamic UI creation.
- Optimized with Vite for fast loading and execution.

## Installation:

1. Clone the repository:

    ```
    git clone https://github.com/yourusername/react-vite-datepicker.git
    ```

2. Install dependencies:

    ```
    cd react_date_datepicker
    npm install
    ```

## Usage:

1. Run the project:

    ```
    npm run dev
    ```

2. Open your browser and navigate to:

    ```
    http://localhost:3000
    ```
